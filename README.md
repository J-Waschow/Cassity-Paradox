# Cassity-Paradox

**An unbiased pre-battle military outcome prediction algorithm.**

A comparative machine learning project that predicts interstate war outcomes using **only pre-engagement factors** from the Correlates of War (COW) dataset. Multiple model versions were iteratively developed and evaluated.

## Objective
Develop a transparent, bias-mitigated framework for forecasting war winners based exclusively on quantifiable pre-battle variables (force ratios, logistics, industrial capacity, leadership metrics, geography, etc.).
